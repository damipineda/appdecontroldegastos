Voy a crear una aplicación web completa de control de gastos desde cero.

# Estructura del Proyecto
*   `index.html`: La estructura principal usando Bootstrap 5 para un diseño responsivo y moderno.
*   `css/style.css`: Estilos personalizados para mejorar la apariencia.
*   `js/app.js`: Lógica de la aplicación (CRUD de gastos y persistencia en LocalStorage).

# Funcionalidades Principales
1.  **Formulario de Registro**: Campos para Concepto, Monto y Categoría.
2.  **Listado de Gastos**: Tabla dinámica que muestra los gastos registrados.
3.  **Persistencia de Datos**: Uso de `localStorage` para guardar los datos como JSON, asegurando que no se pierdan al recargar.
4.  **Cálculo de Total**: Visualización en tiempo real del total de gastos.
5.  **Gestión**: Posibilidad de eliminar gastos individuales.

# Tecnologías
*   **HTML5**: Estructura semántica.
*   **Bootstrap 5**: Framework CSS para estilos rápidos y responsivos (usaré CDN).
*   **JavaScript (ES6+)**: Lógica del cliente y manipulación del DOM.
